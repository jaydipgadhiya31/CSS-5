    why CSS?

    • CSS saves time: You can write CSS once and reuse the same sheet in multiple HTML pages.
    • Flexibility: It offers a wide range of styling options, from basic properties like color and font size to advanced techniques like animations and transformations, giving designers and developers the flexibility to create visually appealing and interactive web experiences.
    • Efficiency: By using external style sheets, CSS enables faster loading times since the browser can cache stylesheets, reducing the need to download styling information repeatedly for each page.
    • Separation of Concerns: CSS separates the content (HTML) of a web page from its presentation, making it easier to maintain and update both aspects independently.
